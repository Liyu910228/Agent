# MCP 参考

## MCP 工具元数据

每个 MCP 工具建议包含：

- id
- name
- description
- inputSchema
- permissionScope
- enabled

## 第一版模拟 MCP 工具

- search_docs：模拟文档检索
- query_order_status：模拟订单状态查询
- create_ticket：模拟工单创建

## 后续真实 MCP 接入

后续接入真实 MCP Client 时，需要补充：

- MCP server 配置方式
- 工具发现流程
- 工具权限控制
- 工具调用超时
- 工具调用错误处理
